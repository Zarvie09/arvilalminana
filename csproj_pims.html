Parish Data Certificate System

A centralized web-based application inspired by the UI of a Blazor app, designed using Tailwind CSS. It manages sacramental records such as:

  • Baptismal
  • Confirmation
  • Marriage
  • Death

Features include record searching, certificate printing, and responsive UI components styled with Bootstrap for a clean, Blazor-like experience.

Baptismal Records

Manage baptismal entries and print certificates.

Confirmation Records

Store confirmation details and generate documents.

Marriage Records

Track and update marriage documentation.

Death Records

Log and print certificates for deceased parishioners.

Search & Reports

Advanced search for records with export and printing capabilities.

Technologies Used

  • Programming Languages: C#, Razor, HTML, JavaScript
  • Frameworks & Tools: .NET MAUI Blazor Hybrid (modern version), Windows Forms with RDLC (.NET Framework - legacy version), Bootstrap (for UI)
  • Database: Microsoft SQL Server

Ongoing Development & Enhancements

  • Implementing role-based access control and user authentication
  • Adding cloud-based database synchronization for parish networks
  • Enhancing UI with animations and certificate preview templates
  • Integrating audit trail logs and backup/restore capabilities
